## Quick Start

#### Install dependencies

```
$ python3 -m venv .virtualenv
$ ./ve pip install -r requirements.txt
$ ./ne yarn install
```

#### Run up for development

- front end
```
$ make fe
```

- python
```
$ make web
```

> visit [http://127.0.0.1:5000](http://127.0.0.1:5000) for website.


#### Run up for production

```
$ make prd
```
